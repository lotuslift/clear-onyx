# Clear² Onyx

**Clear_Onyx × Residual Recognition — Signal Adapter Lab**

Clear² Onyx is a self-contained browser laboratory for inspecting several
orthogonal reads of one shared state:

- a continuous disc operator and spectral surface;
- a complex phase probe;
- a six-direction support reader, S₆;
- an EMA session-memory deformation, μ;
- a verified D₄ landmark auditor, R²;
- a signal-adapter interface for mirror-return comparison.

## Live laboratory

Open the GitHub Pages deployment linked in this repository's About section.

## Running locally

Download the repository and open `index.html` in a modern browser.

No installation, build step, framework, package manager, or external
dependency is required.

## Architectural boundary

The surface renderer, complex-plane reader, S₆ reader, R² landmark auditor,
and signal adapters are separate reads or audits. They do not feed back into
one another unless that dependency is explicitly declared in the source.

The R² module accepts only manually selected, independently verified D₄
landmarks. It does not quantize the continuous Clear_Onyx transport.

Session memory μ is an EMA circular-mean memory used to deform the pupil.
It is not the central residue λ reserved by the associated
return/residue/transport work.

## Current status

This repository contains an exploratory reference implementation and
interactive demonstration. It should not be read as a general proof connecting
the continuous carrier, S₆ reader, D₄ auditor, or every possible signal client.
